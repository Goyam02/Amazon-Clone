About Amazon.in Clone

This is a clone of Amazon.in built with Flask, SQLAlchemy, and Bootstrap.

This project is created for educational purposes to demonstrate web development skills using Python and the Flask framework. It mimics the functionality and design of the popular e-commerce platform Amazon.

Features

  • User authentication (register, login, profile management)
  • Product browsing by category
  • Product search functionality
  • Product detail pages with reviews
  • Shopping cart
  • Checkout process
  • Responsive design similar to Amazon
